# Download Open Grind

Open Grind is in active development. [Contribute to the project](https://git.opengrind.org/open-grind/open-grind/) or [join the discussion](https://matrix.to/#/#opengrind:opengrind.org) to help us prioritize features and improvements.

As of May 21th, 2026: **🚧 Early MVP testing build is available 🚧**.

Download the latest version from [releases](https://git.opengrind.org/open-grind/open-grind/releases). All releases are signed. Never install Open Grind from unofficial sources. Verify [PGP](https://opengrind.org/pgp) signature before installing.
