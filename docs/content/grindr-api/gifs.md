# GIFs, WIP

## Get trending gifs, WIP

```
GET /v1/gifs/trending
```

Query (optional):

- `limit` — integer, optional
- `rating` — string, optional

## Search gifs, WIP

```
GET /v1/gifs/search
```

Query (optional):

- `limit` — integer, optional
- `rating` — string, optional
- `q` — string, optional
- `offset` — integer, optional
